Funciones

Bloques de código reutilizables. Tipos: declaradas, expresadas, arrow functions (ejemplos en texto).

Función declarada

function saludar(nombre) {
  return 'Hola ' + nombre;
}
console.log(saludar('Ana'));

Función expresada y flecha

// Función expresada
const suma = function(a, b) {
  return a + b;
};

// Arrow function
const resta = (a, b) => a - b;
