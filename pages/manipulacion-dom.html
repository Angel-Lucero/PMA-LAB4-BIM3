Manipulación del DOM

El DOM (Document Object Model) es la representación en memoria de la estructura de un documento HTML. JavaScript nos permite manipularlo para cambiar el contenido, estilos y estructura de una página en tiempo real.

Seleccionar elementos

Los métodos más comunes para seleccionar elementos son:

  • getElementById(id) — Selecciona un elemento por su atributo id.
  • querySelector(selector) — Selecciona el primer elemento que coincida con un selector CSS.
  • querySelectorAll(selector) — Selecciona todos los elementos que coincidan con un selector CSS.

Ejemplo


// Seleccionando por ID
const titulo = document.getElementById("miTitulo");

// Seleccionando por selector CSS
const parrafo = document.querySelector(".texto");

// Cambiando el contenido
titulo.innerHTML = "Nuevo título dinámico";

Modificar contenido

La propiedad innerHTML permite cambiar el HTML interno de un elemento.

Ejemplo


document.getElementById("demo").innerHTML = "Texto cambiado desde JavaScript";
