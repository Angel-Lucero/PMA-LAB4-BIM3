Operadores

Tipos principales

  • Aritméticos: + - * / %
  • Asignación: =, +=, -=
  • Comparación: ==, ===, !=, !==, >, <
  • Lógicos: && || !

Ejemplos (solo texto)

// Aritméticos
let a = 5 + 3; // 8
let b = 10 % 3; // 1

// Comparación
2 == '2'   // true (con conversión)
2 === '2'  // false (sin conversión)
