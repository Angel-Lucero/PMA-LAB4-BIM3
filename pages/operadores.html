Operadores en JavaScript

Los operadores son símbolos que permiten realizar operaciones sobre valores y variables. Se clasifican según su función, ya sea aritmética, asignación, comparación o lógica.

Operadores aritméticos

Se utilizan para realizar cálculos matemáticos entre valores numéricos.

let a = 5 + 3;  // suma → 8
let b = 10 - 2; // resta → 8
let c = 4 * 2;  // multiplicación → 8
let d = 16 / 2; // división → 8
let e = 10 % 3; // módulo → 1

Operadores de asignación

Permiten asignar valores a variables y combinarlos con operaciones aritméticas.

let x = 5;     // asignación
x += 3;          // equivalente a x = x + 3 → 8
x -= 2;          // equivalente a x = x - 2 → 6
x *= 2;          // equivalente a x = x * 2 → 12
x /= 3;          // equivalente a x = x / 3 → 4

Operadores de comparación

Se utilizan para comparar valores y devuelven true o false.

2 == '2'   // true  (igualdad con conversión)
2 === '2'  // false (igualdad estricta)
5 != 3     // true  (diferente)
5 !== 5    // false (diferente estricta)
4 > 3      // true
3 < 4      // true

Operadores lógicos

Permiten combinar expresiones booleanas.

let a = true && false; // false (AND)
let b = true || false;   // true  (OR)
let c = !true;           // false (NOT)
