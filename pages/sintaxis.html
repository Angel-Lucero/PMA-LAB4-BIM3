Sintaxis básica

Reglas y conceptos básicos de la sintaxis en JavaScript.

Reglas importantes

  • JavaScript distingue mayúsculas y minúsculas.
  • Comentarios: // (línea) y /* ... */ (bloque).
  • El punto y coma es opcional, pero se recomienda por claridad.

Ejemplo (solo texto)

// Comentario de una línea
            let texto = "Hola mundo";
            console.log(texto);

            /* Comentario de bloque */
            let numero = 5;
            console.log(numero + 2);
