Sintaxis básica en JavaScript

Reglas y conceptos fundamentales de la sintaxis en JavaScript que facilitan la escritura de código limpio y funcional.

Reglas importantes

  • JavaScript distingue entre mayúsculas y minúsculas.
  • Comentarios: // (línea) y /* ... */ (bloque).
  • El punto y coma es opcional, pero se recomienda usarlo por claridad.

Ejemplos

// Comentario de una línea
let texto = "Hola mundo";
console.log(texto);

/* Comentario de bloque */
let numero = 5;
console.log(numero + 2);
